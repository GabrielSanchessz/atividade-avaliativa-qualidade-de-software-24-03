## 🚀 Quick Start

```bash
# 1. Instalar dependências
npm install

# 2. Iniciar desenvolvimento com live reload
npm run live

# 3. Fazer mudanças e ver refletir instantaneamente!
# Edite qualquer arquivo em src/ e salve (Ctrl+S)
```
